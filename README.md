# FrameStudio 📸
A clean & modern responsive landing page for a photography, Animation studio.

**🔴 Live Demo:** https://rutucodeart.github.io/FrameStudio/

### ✨ Features
- Fully Responsive Design – Mobile, Tablet, Desktop Working
- Modern UI with Hero Section, Gallery, Services & Contact Form
- Built with Mobile-First Approach
- Clean code structure for easy updates

### 🛠️ Built With
- **Frontend:** HTML, CSS
- **Tools:** VS Code, GitHub

### 🎯 What I Learned
- How to convert Figma design into pixel-perfect HTML/CSS
- Using CSS Flexbox for responsive layouts
- Making websites that look good on all screen sizes
